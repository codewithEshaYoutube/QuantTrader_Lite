Features

Multi-source price data
CoinGecko → Binance → CryptoCompare fallback chain. Never fails silently.
Trend-aware AI signals
Gemini 2.0 Flash reads live price + session trend to generate BUY / SELL / HOLD with a reason.
Full trade log
Every signal and paper trade logged to JSON. Charts, history table and signal distribution included.
4 coins supported
Bitcoin, Ethereum, Solana, BNB — each with their own chart tab and Kraken pair.
Hackathon-ready
Clean live demo with one command. Explainable AI reasoning visible to judges on every trade.
Auto-refresh mode
Toggle auto-refresh to run the full agent loop every 60 seconds, hands-free.
